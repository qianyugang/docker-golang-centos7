# docker-golang-centos7

## describe

Golang Docker environment based on centos7

## usage

### start

```
docker-compose up -d
```

### stop

```
docker-compose stop
```

### exec

```
docker-compose exec app /bin/bash
```

### host & port

```
localhost:8888
```

## notice

- centos version : 7
- golang version : 1.17.5
